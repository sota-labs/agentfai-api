# AgentFai

## Dependencies
NodeJS v18

## Install dependencies
```bash
npm install
```

## Run script
```bash
npm run start:dev api
```

## Process description

- `api` - API server

## Create indexes

```bash
node dist/scripts/000-create-indexes.js up
```

## Drop indexes
```bash
node dist/scripts/000-create-indexes.js down
```
